   バージョン: SimpleWeightManager 1.0.0
      著作者 : Yor-Jihons

         用途:
               体重の推移を可視化する。

  できること :
               @ 体重データの追加
               @ 体重の推移の可視化
               @ 体脂肪率データの追加
               @ 体脂肪率の推移データの可視化
               @ 体重データ等の一括削除

インストール :
               このzipファイルを解凍すれば完了

    動作環境 :
               @ Windows 7以降

    開発環境 :
               @ Windows 11
               @ C#.NET (.NET6)
